# 🚀 Suite de Gestión Logística en TypeScript

Esta es una colección de soluciones modulares desarrolladas en **TypeScript** para la automatización de procesos de facturación, gestión de inventarios y cálculos de beneficios sociales.

## 🛠️ Sistemas Incluidos

| Sistema | Funcionalidad Principal | Lógica de Negocio |
| :--- | :--- | :--- |
| **Alquiler Pro** | Control de horas alquiladas. | Descuento del 30% tras 12 horas. |
| **Burger Palace** | Gestión de pedidos de comida rápida. | Cálculo dinámico de combos y totales. |
| **Subsidios Armenia** | Programa de bienestar social. | Segmentación por edad (Senior/Base). |
| **Parqueadero 24/7** | Control de flujo vehicular. | Tarifas por tipo y descuento por estadía larga. |
| **Bibliotech** | Administración de multas bibliotecarias. | Recargos escalonados por días de retraso. |

---

## 📋 Características Técnicas

El proyecto implementa las mejores prácticas de desarrollo moderno:
* **Tipado Fuerte:** Uso estricto de `interfaces` y `enums` para evitar errores en tiempo de ejecución.
* **Clean Code:** Lógica de negocio separada de la lógica de presentación.
* **Formateo Localizado:** Uso de `toLocaleString('es-CO')` para representación correcta de moneda.
* **Escalabilidad:** Configuración basada en constantes para ajustes rápidos de precios o políticas.
